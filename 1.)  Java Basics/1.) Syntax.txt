
+ Variables Syntax: variableType variableName = variableValue;
- Example: int age = 22;

+ Variable Types:
- String: Stores text
- int: Stores integers
- float: Stores decimals
- char: Stores single character
- boolean: stores true/false values

+ Declaring Multiple Variables: int variableName = variableValue, int variableName = variableValue;
- Example: int x = 5, int y = 6, int z = 7;

+ Declaring One Value To Multiple Variables: 
- Example:	
	String x, y, z;
	x = y = z = "Hello World";

+ Declaring Constants: final variableType variableName = variableValue;
- Example: final int age = 22;

+ Console Log: System.out.println("Hello World");

* Statements must be terminated with ;
* Single characters must be enclosed with ''
* Strings must be enclosed with ""

+ Arithmetic Operators:
- Addition: +
- Subtraction: -
- Multiplication: *
- Division: /
- Modulo (Remainder after division): %

+ Assignment Operators:
- = 
- += 
- -=
- *=
- /=
- %=

+ Relational Operators:
- Equal To: ==
- Not Equal To: !=
- Greater Than: >
- Less than: <
- Greater Than or Equal To: >= 
- Less Than or Equal To: <=

+ Logical Operators:
- Logical AND: &&
- Logical OR: ||
- Logical NOT: !

+ Unary Operators:
- Plus: +
- Minus: -
- Increment: ++
- Decrement: --
- Logical Complement Operator: !
